# Yandex Maps Hammer PRO (Source Engine)

Профессиональный инструмент для мапперов на движке **Source** (Half-Life 2, CS:GO, TF2, Garry's Mod).
Позволяет измерять расстояния и углы на Яндекс Картах в игровых единицах (**Hammer Units**).

![Preview](preview.png)
*> На скриншоте: панель управления, конвертация расстояния в юниты (u) и расчет угла.*

## 🎮 Зачем это нужно?
Движок Source использует свою систему измерений. Этот скрипт помогает переносить реальные объекты в игру с точностью до юнита.

> **Формула:** 1 метр = **39.37** хаммер-юнитов (hu).  
> *Источник: [Valve Developer Wiki - Dimensions (HL2/CS:S)](https://developer.valvesoftware.com/wiki/Dimensions_(Half-Life_2_and_Counter-Strike:_Source))*

## 🚀 Установка (2 минуты)

1. Установи расширение **[Tampermonkey](https://www.tampermonkey.net/)** для своего браузера.
2. Нажми на ссылку ниже:
   
   👉 **[УСТАНОВИТЬ СКРИПТ](https://raw.githubusercontent.com/postal1443/yandex-maps-to-hammer-units/main/user.js)**

## ⚙️ Функции версии 4.8
- 📏 Конвертация метров в юниты.
- 📐 Измерение углов (0-90° / 0-360°).
- 🕸 Настройка сетки (Grid) и шага угла.
- 🖱 Перетаскиваемая панель.
- 🧹 Hard Reset.
- ⚡ Оптимизация (спящий режим при бездействии).

## 📸 Как использовать
1. Открой [Яндекс Карты](https://yandex.ru/maps).
2. Включи инструмент "Линейка".
3. Протяни линию — появятся зеленые цифры (`📏 128u`, `📐 45°`).
