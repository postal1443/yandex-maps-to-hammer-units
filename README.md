# Yandex Maps Hammer PRO (Source Engine)

Профессиональный инструмент для мапперов на движке **Source** (Half-Life 2, CS:GO, TF2, Garry's Mod).
Позволяет измерять расстояния и углы на Яндекс Картах в игровых единицах (**Hammer Units**).

## 🎮 Зачем это нужно?
Движок Source использует свою систему измерений. Этот скрипт помогает переносить реальные объекты в игру с точностью до юнита.

> **Формула:** 1 метр = **39.37** хаммер-юнитов (hu).  
> *Источник: [Valve Developer Community - Map Units](https://developer.valvesoftware.com/wiki/Map_Units)*

## 🚀 Установка (2 минуты)

1. Установи расширение **[Tampermonkey](https://www.tampermonkey.net/)** для своего браузера (Chrome, Firefox, Edge).
2. Нажми на ссылку ниже, чтобы автоматически установить скрипт:
   
   👉 **[УСТАНОВИТЬ СКРИПТ](https://raw.githubusercontent.com/postal1443/yandex-maps-to-hammer-units/main/user.js)**
   
   *(Если ссылка не сработала: скачай файл `user.js` из этого репозитория вручную и перетащи его в окно управления Tampermonkey).*

## ⚙️ Функции версии 4.8
- 📏 **Конвертация метров в юниты** (с учетом шага сетки).
- 📐 **Измерение углов** (режимы 0-90° и 0-360°, горизонталь/вертикаль).
- 🕸 **Настройка сетки** (Grid) и шага угла (Angle Step).
- 🖱 **Перетаскиваемая панель** (можно двигать по экрану).
- 🧹 **Hard Reset** (полный сброс линейки одной кнопкой).
- ⚡ **Оптимизация** (умный режим ожидания, нагрузка на CPU почти нулевая).

## 📸 Как использовать
1. Открой [Яндекс Карты](https://yandex.ru/maps).
2. Включи инструмент "Линейка" (значок линейки в меню слева).
3. Появится черная панель **HAMMER PRO**.
4. Протяни линию на карте — внизу измерения появятся зеленые цифры:
   - `📏 128u` — расстояние в юнитах (округленное по сетке).
   - `📐 45°` — угол наклона линии.

## 🛠 Технические детали
- Скрипт использует `requestAnimationFrame` и отслеживание активности мыши для экономии ресурсов.
- Работает только на домене `yandex.ru/maps`.
- Не требует перезагрузки страницы.
